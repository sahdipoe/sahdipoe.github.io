Langkah-langkah Upload Website ke GitHub Pages

A. Buat Akun GitHub

Jika Anda belum memiliki akun GitHub, langkah pertama yang harus dilakukan adalah membuat akun. Kunjungi situs GitHub dan klik tombol "Sign up". Isi informasi yang diperlukan seperti nama pengguna, email, dan kata sandi. Ikuti instruksi hingga pendaftaran selesai.

B. Siapkan Repository di GitHub

  • Masuk ke Akun GitHub Anda: Login menggunakan akun Anda.
  • Buat Repository Baru: Klik tombol "New", beri nama repository, pilih "Public", lalu klik "Create Repository".

C. Persiapkan File Website Anda

Pastikan semua file website, seperti HTML, CSS, dan JavaScript, tersusun dengan rapi dalam struktur folder.

D. Upload File Website ke Repository GitHub

  1. Buka repository yang telah dibuat.
  2. Klik "Add File" dan pilih "Upload Files".
  3. Upload semua file website, termasuk index.html.
  4. Isi deskripsi "Commit changes" dan klik "Commit changes".

E. Aktifkan GitHub Pages

  1. Buka tab "Settings" di repository Anda.
  2. Gulir ke bagian "GitHub Pages".
  3. Pilih "main branch" sebagai sumber, lalu klik "Save".

F. Akses Website Anda

Setelah GitHub Pages aktif, akses website Anda melalui URL seperti:

https://username.github.io/nama-repository/

Gantilah username dan nama-repository sesuai pengaturan Anda.
