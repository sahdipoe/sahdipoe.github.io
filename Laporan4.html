Pembahasan

1. Dari Manakah Mikrotik Berasal?

Mikrotik berasal dari Latvia, sebuah negara di Eropa Utara. Perusahaan ini didirikan pada tahun 1996 dan terkenal dengan produk-produk jaringan seperti router dan perangkat lunak manajemen jaringan.

2. Tipe Mikrotik yang Dipakai di Lab

Tipe Mikrotik yang dipakai adalah Mikrotik RB391-2nd. Tipe ini merupakan salah satu varian RouterBoard yang sering digunakan untuk pengujian dan pengelolaan jaringan di lingkungan laboratorium atau pembelajaran.

3. Parameter pada Halaman Awal Winbox

  • MAC Address: Alamat fisik perangkat Mikrotik.
  • IP Address: Alamat IP yang terkonfigurasi pada perangkat.
  • Identity: Nama atau identitas perangkat Mikrotik.
  • Version: Versi firmware atau RouterOS yang digunakan.
  • Uptime: Waktu perangkat berjalan sejak terakhir kali di-restart.

4. Menu yang Ada di Winbox

  • Interfaces: Konfigurasi port jaringan.
  • IP: Pengaturan IP Address, Routes, Firewall, DNS, dll.
  • Bridge: Pengaturan bridge jaringan.
  • Wireless: Pengaturan jaringan nirkabel.
  • Queue: Manajemen bandwidth atau antrean lalu lintas jaringan.
  • Routing: Pengaturan routing di jaringan.
  • System: Pengaturan sistem seperti identitas, waktu, reboot, dll.
  • Tools: Alat untuk manajemen dan monitoring jaringan seperti ping, traceroute, dll.
  • Logs: Menampilkan log aktivitas sistem.

5. Konfigurasi Mikrotik yang Diketahui

  • IP Address Configuration: Menetapkan alamat IP pada antarmuka router.
  • NAT: Untuk menerjemahkan alamat IP dari jaringan lokal ke publik.
  • Firewall Configuration: Membuat aturan keamanan jaringan.
  • DHCP Server Configuration: Mengatur router sebagai server DHCP.
  • Static Route Configuration: Menambahkan rute manual untuk lalu lintas tertentu.
  • Wireless Configuration: Mengonfigurasi router sebagai access point atau client nirkabel.
  • Bandwidth Management: Mengatur pembatasan kecepatan atau prioritas lalu lintas jaringan.
