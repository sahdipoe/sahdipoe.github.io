Pembahasan

1. Topologi dan Persiapan Awal

Sebelum masuk ke konfigurasi, kita perlu menentukan topologi jaringan dan alamat IP yang akan digunakan.

  • MikroTik 1 (R1):
    • Interface ether1: 10.10.10.1/24
    • Interface ether2: 192.168.100.1/24
  • MikroTik 2 (R2):
    • Interface ether1: 10.10.10.2/24
    • Interface ether2: 11.11.11.1/24
    • Interface ether3: 192.168.200.1/24
  • MikroTik 3 (R3):
    • Interface ether1: 11.11.11.2/24
    • Interface ether2: 192.168.50.1/24

2. Langkah Konfigurasi MikroTik

2.1. Konfigurasi IP Address

Berikut ini langkah-langkah konfigurasi IP address pada setiap MikroTik:

  • MikroTik 1 (R1): Interface ether1: 10.10.10.1/24, ether2: 192.168.100.1/24
  • MikroTik 2 (R2): ether1: 10.10.10.2/24, ether2: 11.11.11.1/24, ether3: 192.168.200.1/24
  • MikroTik 3 (R3): ether1: 11.11.11.2/24, ether2: 192.168.50.1/24
2.2. Konfigurasi Static Routing

Tambahkan routing statis untuk memastikan komunikasi antar router:

  • R1: dst. 192.168.2.0/24 → gw 11.11.11.1
  • R2: dst. 192.168.1.0/24 → gw 192.168.12.1
  • R3: dst. 192.168.2.0/24 → gw 192.168.200.1
2.3. Testing Koneksi

Gunakan perintah ping untuk menguji konektivitas antar router.
